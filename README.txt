はじめに
========

　90年代前半、当時のPC性能が低かったため、エミュレータは、アセンブラ言
語で記述されていました。その後、90年代後半になると、エミュレータは、主
に、C言語で記述されるようになり、性能的に厳しい時だけ、アセンブラ言語
が併用されるようになりました。2000年代前半には、実験的な試みとして、
Javaによるエミュレータが開発され、ウェブ上でエミュレータを動作させるこ
とに成功しました。そして、2000年代後半には、Javaでの実装を基盤として、
DoCoMo社の携帯電話（iモード）上で動作するエミュレータが出現しました。

　Javaは、仮想マシン上で動作することを前提としていますが、変数には型が
ある本格的な言語です。一方、Pythonは、Perl、Rubyと同じ、いわゆるスクリ
プト言語で、変数には型がありません。スクリプト言語でも、エミュレータが、
実装できるのでしょうか？

　PyChip8は、「Pythonで（ゲーム機）エミュレータが記述できるか？」とい
う問いに答えるために、実験的に実装したものです。そして、その答えは、
「CHIP8レベルであれば、動作するものが実装できる」というものでした。ま
た、GameBoy(TM)程度であれば、実装できるのではないかという感触が得られ
ました。

起動方法
========

(a) Python3.7（python-3.7.2.exe）を実行、インストールします
　　（https://www.python.org/downloads/より取得）
(b) コマンドラインより、Python上のSDLラッパPyGameを実行、インストールします
　　    C> python -m pip install pygame

(c) Chip8ゲームパック（c8games.zip）アーカイブを解凍します
　　（http://www.zophar.net/roms.phtml?op=show&type=chip8より取得）
(d) コマンドラインから、以下のように実行します

　　    C> python PyChip8.py <ROMファイル名>
　　例：C> python PyChip8.py c8games/PONG

(e) 終了時には、[x]をクリックします


操作方法
========

キーは、以下のようにマッピングされてます。

オリジナル　|1|2|3|C|　　マッピング　|4|5|6|7|
　　　　　　|4|5|6|D|　　　　　　　　|R|T|Y|U|
　　　　　　|7|8|9|E|　　　　　　　　|F|G|H|J|
　　　　　　|A|0|B|F|　　　　　　　　|V|B|N|M|


仕様
====

メモリ
------
　- ROMファイル（200H - F10H）
　- 16進数フォント（F10H - F60H）

レジスタ
--------
　- データレジスタ（V0 .. VF）
　- アドレスレジスタ（I）
　- タイマ（ディレイ、サウンド）
　- スタック（16ワード長、スタックポインタ）

グラフィックス
--------------
　- スプライト（CHIP-8モード：8 x 1 .. 15）
　- 衝突フラグ
　- 16進数フォント

命令セット
----------
　- CHIP-8命令（代入、算術、条件分岐、サブルーチン、スプライト、等）

キーボード
----------
　- 16進数キーボード
